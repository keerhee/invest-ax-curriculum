Workflow

워크플로우 흐름 (질의)

🔴 Trigger→ Webhook — 자연어 질의 + ticker 필터
🟣 Embed→ OpenAI 질의 임베딩
🟢 Vector→ Supabase top-20 cosine search (필터: ticker)
🟢 BM25→ Supabase ts_rank — 키워드 매칭 top-20 (병렬)
⚫ Merge→ Code 노드 — 두 결과 dedup, 합집합 ~30건
🔵 Rerank→ Cohere Rerank — top 5 재순위
🟣 Answer→ Claude Sonnet — top-5를 컨텍스트로 인용 답변
🟢 Reply→ Slack — 답변 + [출처1, 출처2] 링크
Schema

Supabase 테이블 (W5 확장)

create table disclosures (
  id bigserial primary key,
  ticker text not null,
  doc_type text,           -- '사업보고서'|'분기'|'주요공시'
  doc_year int,
  page int,
  section text,
  content text,
  embedding vector(1536),
  fts tsvector              -- 한국어 BM25용
);

-- 인덱스
create index on disclosures using ivfflat (embedding vector_cosine_ops);
create index on disclosures using gin (fts);

-- Trigger로 fts 자동 생성
create function disclosures_fts_trigger() returns trigger as $$
begin
  new.fts := to_tsvector('simple', new.content);
  return new;
end $$ language plpgsql;

create trigger disclosures_fts_update before insert or update
on disclosures for each row execute function disclosures_fts_trigger();

인용 강제 프롬프트

너는 IR 자료 분석가다. 아래 검색 결과만으로 답변하고, 반드시 출처를 인용하라.

질문: {{사용자 질문}}

검색 결과 (top 5, 신뢰도 순):
[1] {ticker} {doc_type} p.{page}: {content}
[2] ...

응답 규칙:
- 검색 결과에 명시된 사실만 사용. 외부 지식 금지.
- 모든 주장에 [숫자] 형식 출처 인용.
- 검색 결과로 답할 수 없으면 "근거 부족" 명시.
